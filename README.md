I  created simple messenger in terminal. It is likely possible that in future I will add more features but for now this code allows to:
- log in and out
- create accounts with different privileges (admin or normal user)
- create account from main panel or from admin account (admin account can be created only directly from database or admin account
- send messages to particular user and read your messages from chats you participle with information who and when sent a message
-remove your account or delete one from admin account
